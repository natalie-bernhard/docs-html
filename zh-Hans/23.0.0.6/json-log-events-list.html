JSON 日志事件参考列表

Open Liberty 从服务器运行时和应用程序生成 JSON 日志记录事件。 您可以使用这些事件来收集和分析有助于更好地了解应用程序行为的数据。

Open Liberty生成以下类型的事件:

  • 消息事件

  • 跟踪事件

  • FFDC 事件

  • HTTP 访问事件

  • 审计事件

支持的审计事件及其审计数据

Open Liberty Audit 功能部件可捕获包含来自服务器运行时环境和应用程序的安全详细信息的可审计事件。 您可以使用从审计事件生成的数据来分析配置的环境。

Open Liberty 可以生成 JSON 或 CADF 格式的审计事件。 将以以下 JSON 格式类型捕获审计事件，以帮助确定可以改进所配置环境的不同区域:

  • 管理审计服务 (SECURITY_AUDIT_MGMT)

  • SCIM 操作/成员管理 (SECURITY_MEMBER_MGMT)

  • Servlet 3.0 API: 登录/认证 (SECURITY_API_AUTHN)

  • Servlet 3.0 API: 注销 (SECURITY_API_AUTHN_TERMINATE)

  • 表单注销 (SECURITY_AUTHN_TERMINATE)

  • 基本认证 (SECURITY_AUTHN)

  • 客户机证书认证 (SECURITY_AUTHN)

  • 表单登录认证 (SECURITY_AUTHN)

  • Servlet runAs 授权 (SECURITY_AUTHN_委派)

  • EJB 委派 (SECURITY_AUTHN_委派)

  • 故障转移到基本认证 (SECURITY_AUTHN_FAILOVER)

  • 不受保护的 servlet 授权 (SECURITY_AUTHZ)

  • JACC Web 授权 (SECURITY_AUTHZ)

  • JACC EJB 授权 (SECURITY_AUTHZ)

  • EJB 授权 (SECURITY_AUTHZ)

  • JMS 认证 (SECURITY_JMS_AUTHN)

  • JMS 授权 (SECURITY_JMS_AUTHZ)

  • SAF 授权服务 API 请求 (SECURITY_SAF_AUTHZ)

  • SAF 授权异常 (SECURITY_SAF_AUTHZ_DETAILS)

  • JMX MBean 注册 (JMX_MBEAN_REGISTER)

  • JMX MBean 操作 (JXM_MBEAN)

  • JMX MBean 属性操作 (JMX_MBEAN_ATTRIBUTES)

  • JMX 通知 (JMX_NOTIFICATION)

JMX_NOTIFICATION

您可以使用 JMX_NOTIFICATION 事件来捕获 JMX 通知中的审计信息。 下表提供了 JMX_NOTIFICATION 事件的字段以及每个字段的描述:

JMX_NOTIFICATION 事件字段
FIELD 描述

类型

用于标识事件类型的字符串。

主机

作为事件源的服务器的主机名。

ibm_userDir

作为事件源的服务器的用户目录。

ibm_serverName

作为事件源的服务器的名称。

ibm_datetime

事件发生的时间。

ibm_sequence

事件的序号，这对于排序具有相同时间戳记的记录很有用。

ibm_threadId

日志行中的线程标识，例如 00000015。 线程标识是字符串而非数字。

ibm_audit_eventName

审计事件的名称。

ibm_audit_eventSequence编号

审计事件的序号。

ibm_audit_eventTime

事件发生的时间。

ibm_audit_observer.id

事件的观察者的标识。

ibm_audit_observer.name

事件的观察者的名称: JMXService。

ibm_audit_observer.typeURI

事件观察者的唯一 URI: service/server。

ibm_audit_Outcome

活动的结果。

ibm_audit_reason.reasonCode

指示结果的底层成功或错误代码的值。 通常，值为 200 表示成功。

ibm_audit_reason.reasonType

指示与请求关联的底层机制 (例如 HTTP ， HTTPS ， JMS 或 EJB) 或结果背后的状态的值。

ibm_audit_target.id

操作目标的标识。

ibm_audit_target.jmx.mbean.action

正在对 MBean 属性执行的 MBean 操作。

ibm_audit_target.jmx.notification.filter

通知过滤器的名称。

ibm_audit_target.jmx.notification.listener

通知侦听器的名称。

ibm_audit_target.jmx.notification.name

通知的名称。

ibm_audit_target.realm

与目标关联的域名。

ibm_audit_target.typeURI

事件目标的唯一 URI: server/mbean/notification。
