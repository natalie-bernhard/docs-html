安全漏洞 (CVE) 列表

常见漏洞和暴露 (CVE) 系统是由美国国家标准技术研究所 (NIST) 维护的公开已知网络漏洞的参考。 CVE 列表编目了已知的网络安全漏洞。

下表列出了影响 Open Liberty的 CVE ，按修正这些 CVE 的发行版排序。 该表还包含受每个 CVE 影响的 Liberty 功能部件。 您可以通过查看在服务器上安装并运行的功能部件来确定 Liberty 配置是否可能受 CVE 影响。 有时，未在 server.xml 文件中指定的 Liberty 功能部件会由其他功能部件自动启用。 要确认是否在 Liberty 服务器中启用了特定功能部件，请检查 Liberty 服务器的 console.log， messages.log或 trace.log 文件中的 CWWKF0012I 消息。 此消息提供 Liberty 服务器上安装和运行的所有功能部件的综合列表。

CWWKF0012I: The server installed the following features: [appSecurity-2.0, distributedMap-1.0, jndi-1.0, samlWeb-2.0, servlet-3.0, ssl-1.0].

CWWKF0012I 消息使用 "已安装" 一词，但它列出了 Liberty 服务器上已安装和正在运行的功能部件。
